# trps: Bayesian Trophic Position Models using 'Stan'

Bayesian trophic position models using 'stan' by leveraging 'brms' for
stable isotope data. Trophic position models are derived by using
equations from Post (2002)
[doi:10.1890/0012-9658(2002)083\[0703:USITET\]2.0.CO;2](https://doi.org/10.1890/0012-9658%282002%29083%5B0703%3AUSITET%5D2.0.CO%3B2)
, Vander Zanden and Vadeboncoeur (2002)
[doi:10.1890/0012-9658(2002)083\[2152:FAIOBA\]2.0.CO;2](https://doi.org/10.1890/0012-9658%282002%29083%5B2152%3AFAIOBA%5D2.0.CO%3B2)
, and Heuvel et al. (2024)
[doi:10.1139/cjfas-2024-0028](https://doi.org/10.1139/cjfas-2024-0028) .
