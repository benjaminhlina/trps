# Error functions

Error functions

## Usage

``` r
check_column_names(x, arg_name = NULL)

check_data_frame(x, arg_name = NULL)

check_lambda(x, arg_name = NULL)

check_logical(x, arg_name = NULL)

check_numerical(x, arg_name = NULL)
```

## Arguments

- x:

  prior to check @param arg_name the name of the argument to check.

- arg_name:

  the name of the argument to check.
